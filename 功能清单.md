# AI多语种智能导游服务平台 - 后端功能清单

## 项目信息
- **技术栈**: Java 8 + Spring Boot 2.7 + MyBatis-Plus + Session/Cookie + MySQL 8
- **端口**: 8137
- **数据库**: ai_guide_platform
- **文件总数**: 140 个 Java 文件

---

## 一、数据库设计（12 张表）

### 1. sys_user - 用户表
- 字段：id, username, password, nickname, avatar_url, role_code(TOURIST/ADMIN), user_status, email, phone, last_login_time
- 索引：uk_username(唯一), idx_role_code, idx_user_status

### 2. scenic_category - 景点分类表
- 字段：id, category_name, category_desc, sort_no, category_status
- 索引：idx_category_status, idx_sort_no

### 3. scenic_spot - 景点主表
- 字段：id, category_id, spot_name, city, address, longitude, latitude, cover_url, summary, description, open_time, suggest_duration, tips, hot_score, spot_status, creator_id, view_count
- 索引：idx_category_id, idx_city, idx_spot_status, idx_hot_score, idx_spot_name

### 4. scenic_spot_i18n - 景点多语种内容表
- 字段：id, scenic_spot_id, language_code, title, summary, description, tips
- 索引：uk_spot_lang(联合唯一), idx_language_code

### 5. scenic_spot_media - 景点媒体资源表
- 字段：id, scenic_spot_id, media_type, media_url, media_name, sort_no
- 索引：idx_scenic_spot_id, idx_media_type

### 6. guide_route - 路线主表
- 字段：id, route_name, theme, cover_url, summary, description, suggest_duration, suitable_crowd, route_status, creator_id
- 索引：idx_route_status, idx_theme

### 7. guide_route_spot_rel - 路线景点关联表
- 字段：id, route_id, scenic_spot_id, sort_no, stay_duration
- 索引：idx_route_id, idx_scenic_spot_id, uk_route_spot(联合唯一)

### 8. guide_route_i18n - 路线多语种内容表
- 字段：id, route_id, language_code, title, summary, description, travel_tips
- 索引：uk_route_lang(联合唯一), idx_language_code

### 9. user_favorite - 用户收藏表
- 字段：id, user_id, biz_type(SCENIC/ROUTE), biz_id
- 索引：uk_user_biz(联合唯一), idx_user_id

### 10. user_feedback - 用户反馈表
- 字段：id, user_id, feedback_type, content, contact_info, feedback_status, reply_content, reply_time
- 索引：idx_user_id, idx_feedback_status, idx_create_time

### 11. file_resource - 文件资源表
- 字段：id, biz_type, biz_id, file_name, file_url, file_type, file_size, storage_mode, uploader_id
- 索引：idx_biz_type_biz_id, idx_uploader_id

### 12. ai_guide_log - AI问答日志表
- 字段：id, user_id, model_name, biz_type, language_code, scenic_spot_id, route_id, request_summary, response_summary, success_flag, cost_millis, error_message
- 索引：idx_user_id, idx_biz_type, idx_success_flag, idx_create_time

## 十四、启动步骤

### 1. 初始化数据库
```bash
mysql -u root -phcc153502 < src/main/resources/sql/init_database_schema.sql
mysql -u root -phcc153502 < src/main/resources/sql/init_database_data.sql
```

### 2. 启动后端
```bash
cd ai-guide-backend
mvn spring-boot:run
```
访问：http://localhost:8137/doc.html（API文档）

### 3. 启动前端
```bash
cd ai-guide-frontend
npm install
npm run dev
```
